Box Model

Padding

El padding es el espacio entre el contenido del elemento y su borde. Se usa cuando queremos separar el contenido del borde, o aumentar el tamaño interno del elemento.

Border

El border es el literal `borde` de la caja del `elemento`. Se usa principalmente para decorar los elementos.

Margin

El margin es el espacio entre el `borde` del elemento y los `otros elementos`. Se usa para alejar al elemento del resto.
